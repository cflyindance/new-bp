经过对Clover、Square、Snackpass等竞品后台的结构化分析，并结合Gemini、豆包、智谱等多方分类对比，最终形成这份**餐饮POS后台设置项全量归类方案**。方案严格遵循您指定的27个固定一级导航，顺序不变，且未进行任何合并。每个导航下按业务逻辑拆分为合理的二级模块，并列举代表性设置项，覆盖原系统1000+混乱设置项，彻底解决分类不清、层级缺失的问题。

### **分类原则与调整说明**
1.  **硬件与网络彻底分离**：所有涉及物理设备（打印机、钱箱、客显屏、叫号屏、支付终端）及其IP地址、连接方式的配置，统一归入**硬件管理中心**，不再散落于打印、支付等模块。
2.  **财税与支付解耦**：税率、服务费、小费建议归入**支付中心**（面向交易）；对账、结算、BATCH、发票归入**财务中心**（面向后台核算）。
3.  **员工权限与考勤分离**：考勤、工时、工资、排班归入**团队管理**；而敏感操作（如经理强制登出、开钱箱、需要授权才能下的积分菜）的授权和权限控制归入**权限管理中心**。
4.  **菜单与商品划分清晰**：菜品的结构、分类、加料、套餐定义归入**商品中心**；菜单的视觉展示、时段控制、Kiosk菜单布局归入**前厅管理中心**（顾客侧）或**商品中心**（系统级定义）。
5.  **报表与财务独立**：报表模板、展示维度、导出设置归入**报表中心**；资金提现、银行账户、长短款处理归入**财务中心**。
6.  **营销与促销细分**：会员体系、积分、等级归入**会员中心**；折扣规则、满减、优惠券归入**促销中心**；活动整体策划和渠道归入**营销中心**。

## **餐饮POS后台设置项最终全量归类**

### **1. 品牌管理**
- **品牌基础信息**：餐馆名、电话、传真、网站、商家编号、地址、logo、品牌色、版权信息。
- **品牌层级与连锁**：品牌模式（单店/连锁）、品牌作为首页、门店归属品牌组。
- **品牌视觉规范**：打印Logo、小票品牌露出、客显屏品牌封面。

### **2. 门店管理**
- **门店基础信息**：门店名称、营业时间、营业时段、特殊营业日、门店地址、地图定位。
- **门店运营规则**：门店模式（餐厅/快餐）、菜单模式、国家/地区设置。
- **门店数据与维护**：上传地址数据、删除地址数据、清除交易数据、备份数据。

### **3. Dashboard（主页）**
- **看板模块配置**：核心指标显示（销售额、订单数）、快捷入口自定义。
- **主页视觉**：首页视频、背景图、登录引导图、屏保。
- **主页数据刷新**：数据刷新频率、默认加载范围。

### **4. 团队管理**
- **员工档案**：员工姓名、编号、角色、入职/离职状态。
- **考勤与工时**：员工最长工作时间、强制休息时长、自动收工打卡、开工重置时间、下班打卡打印小票、带薪休息时长。
- **排班**：班次模板、第三方排班表链接、同步云员工系统、云员工系统地址。
- **薪酬与小费分配**：工资计算标准、小费计算标准（按净销售额/总销售额）、分摊小费比例、分享小费规则、第三方打卡记录链接。

### **5. 商品中心**
- **菜品结构与属性**：菜单编辑、调味指示、菜品单位、菜序名、成分、按时计价、菜品默认大小。
- **分类与标签**：分类/品类设置（年龄限制、特殊菜单、营业时间）、菜品标签、菜品分装规则。
- **套餐管理**：自定义套餐组合、自动点完套餐、套餐子菜价格展示、套餐单打印数量。
- **价格管理**：默认全局调味价格、总价四舍五入、比价功能。
- **菜单多语言**：组/类/菜/option多语言映射。

### **6. 订单中心**
- **点单流程控制**：跳过选桌/选人数、每单最大人数、点单超时提醒、数量支持小数、减菜后重定向、合单逻辑、分单时展示未分菜品。
- **送厨逻辑**：点击“付款”直接送厨、点击“送厨”整单送厨、结账后自动送厨、打单后自动送厨、未付单直接送厨。
- **单号规则**：最大单号、起始单号、单号模式（时间戳/分类）、单号重置周期、分类单号前缀。
- **订单操作与异常**：删除未付款单、根据菜退款、订单失效原因、删单原因必填、是否通知厨房删单、允许编辑应收金额、取消部分支付订单。
- **订单界面定制**：显示座位、相同菜合并/拆分显示、菜序模式、ASAP时间显示、按键隐藏/显隐（删单、移单、付款等移至“更多”）、点单页配置（整单操作项、菜品详情项）。
- **找单与筛选**：显示总计价格、默认显示未加小费订单、盘点模式。
- **附加费用与折扣**：折扣原因必填、自定义折扣原因、餐段自动加收、合单时自动加收重算、子单促销自动重算、清除线上订单服务费。

### **7. 支付中心**
- **支付方式与通道**：Payment Service选择、微信/支付宝开关、信用卡最小支付金额、支付图标显示。
- **小费流程**：小费算法（固定/百分比）、签字前/后选小费、建议小费百分比、多次支付时基于整单收小费、隐藏现金小费、PINPad上加小费、默认小费值。
- **签名与安全**：签名要求、签名最小金额、签名在PINPad或屏上。
- **税收规则**：加收费用计税、折扣后计税、外卖/外送免税。
- **预授权与BATCH**：结算天数、自动结账时间、BATCH前检查员工打卡、含未付款订单时允许BATCH、未BATCH订单上限、批量结算后打印报告。
- **退款逻辑**：退款审批、退款时支付方式处理。

### **8. 外卖/来取**
- **基础功能开关**：启用网上点餐、自助点餐、网上点餐单号区分。
- **履约选项**：配送/自提/到店取餐、打包展示号码牌、送餐到桌牌号获取方式。
- **订单来源配置**：Kiosk订单类型（堂食/打包）、可自动送厨的Kiosk支付类型。
- **服务端回调**：Online Order Service相关host、merchant ID、port；callback远程与本地地址、延迟时间、加密hash。
- **订餐限制**：收据上打印确认签名栏、免开端口设置。

### **9. 营销中心**
- **营销活动策划**：活动模板、海报（海报Pro）、首页视频。
- **自动化营销**：基于会员标签推送、新客/老客活动。
- **渠道管理**：社交平台链接、二维码点餐入口、第三方平台营销同步。

### **10. 促销中心**
- **折扣体系**：折扣比率设置、折扣原因库、自定义折扣、折扣与其他促销叠加规则。
- **满减/买赠**：促销策略配置、条件与生效时间。
- **抽奖与附加费**：抽奖活动开关、抽取规则；附加费（整单加收、餐具加收、打包带加收、按菜品打包盒加收）。

### **11. 会员中心**

- **会员卡管理**：云端会员模式、会员卡有效期、卡号查询方式、使用模式（仅积分/积分+余额）。
- **会员权益**：会员等级、权益会员介绍、菜单展示积分菜、积分菜兑换限制。

### **12. 礼品卡中心**

### **13. 评价中心**

- **数据维护**：删除菜品评价、评价数据导出。
