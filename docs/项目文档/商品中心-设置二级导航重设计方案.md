### 2.1 套餐点单与展示（`combo-ordering`）

**归类理由**：套餐点完流程与子菜价展示（139/145）；终版归属商品中心 SSOT；POS 点单页展示见前厅 `pos-order-cart`，订单分单/促销见各 hub 文档。

| seq | 功能模块 | 功能设置 | 功能场景描述（摘要） |
|-----|----------|----------|----------------------|
| 139 | 自定义单/套餐 | 自动点完套餐 | — |
| 145 | 价格 | 当套餐价钱规则是可调时,显示套餐子菜价格 | 设置当套餐价钱规则是可调整时,是否显示套餐子菜价格 |

## 3. 落地步骤

1. `node scripts/apply-product-settings-mapping.mjs`
2. `cd admin-web && npm run build:settings-catalog`
3. 商品中心 → 设置 → **套餐点单与展示**

### 3.1 映射表（CSV）

```csv

```

## 4. 边界说明

- 112/146/456/476/593 等在商品管理维护，catalog 不重复展示。
- **139/145** 为套餐点单 SSOT；前厅 `pos-order-cart` 管 POS 行项展示，不重复套餐定义。
- 订单中心管分单/结账/折扣删退；**150 子单促销重算** 见促销中心 `promo-strategy`。
