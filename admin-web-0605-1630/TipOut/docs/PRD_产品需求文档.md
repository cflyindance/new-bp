# TipOut 小费分配系统 - 产品需求文档（PRD）

**文档版本**：v1.1  
**最后更新**：2026-02-27  
**归属品牌**：MenuSifu  
**适用对象**：UI 设计、前端研发、后端研发

---

## 1. 文档概述

### 1.1 产品定位

TipOut 是面向餐饮门店的小费分配（Tip Out）管理工具，隶属于 MenuSifu 产品体系。用于配置小费池计算规则、执行按日期/门店的小费分配、查看与编辑分配明细，并支持导出与邮件发送。

### 1.2 页面清单与依赖关系

| 页面编号 | 页面名称 | 路径 | 需求数量 |
|---------|---------|------|----------|
| **P1** | 小费分配 | index.html | 25 项 |
| **P2** | 小费分配明细 | detail.html | 22 项 |
| **P3** | 小费分配规则 | rules.html | 12 项 |
| **P4** | 新增/编辑规则 | rule-add.html | 28 项 |

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   P3 规则   │ ───► │  P1 分配    │ ───► │ P2 明细     │
│  rule-add   │      │  index      │      │  detail     │
│   P4        │      └─────────────┘      └─────────────┘
└─────────────┘
     │
     └── P3 入口
```

**跨页依赖**：
- P1「分配小费」依赖 P3 存在至少 1 条规则
- P2 明细展示依赖 P1 该日期已完成分配
- P4 为 P3 的新增/编辑入口

| 源页面 | 目标页面 | 依赖逻辑 | 数据/状态 |

| P3/P4 | P1 | 规则存在 | tipout_rules |
| P1 | P2 | 该日期已分配 | tipout_allocated |
| P3 | P4 | 编辑/新增入口 | URL mode&id |
